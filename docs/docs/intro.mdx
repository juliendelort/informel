## Installation

### CDN Installation

Add the following script to your HTML page

```html
<script src="https://unpkg.com/informel"></script>
```

### NPM/Yarn Installation

Install the package:

```bash
npm i informel
yarn add informel
```

Then import informel at the root level of your app:

```javascript
import "informel";
```

## Usage

### Basic Usage

Just wrap your native form in `<inform-el />`:

```html
<inform-el>
    <form>
        <input type="text" name="firstName" placeholder="First name" />
        <input type="text" name="lastName" placeholder="Last name" />
        <button type="submit">Submit</button>
    </form>
</inform-el>
```

Then you can listen to the `submit` event which gives you the form values:

```javascript
document.querySelector("inform-el").addEventListener("submit", ({ detail: { values } }) => {
    console.log("Submitted:", values);
});
```

### Show validation errors

In order to automatically show validation errors, wrap your fields in `<inform-field />`.

```html
<inform-el>
    <form>
        <inform-field>
            <input type="text" name="firstName" placeholder="First name" required />
        </inform-field>
        <inform-field>
            <input type="text" name="lastName" placeholder="Last name" required />
        </inform-field>
        <button type="submit">Submit</button>
    </form>
</inform-el>
```

<iframe width="100%" height="300" src="//jsfiddle.net/juliendelort/5Lfdzsew/4/embedded/result,html/"></iframe>

`informel` leverages [native form validation features](https://developer.mozilla.org/en-US/docs/Learn/Forms/Form_validation)
and displays [native validation messages](https://developer.mozilla.org/en-US/docs/Web/API/HTMLObjectElement/validationMessage) by default.
