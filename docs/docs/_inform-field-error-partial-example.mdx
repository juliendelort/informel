```html
<inform-el>
    <form>
        <label>
            First Name
            <inform-field>
                <input type="text" name="firstName" required />
            </inform-field>
        </label>
        <label>
            Last Name
            <inform-field>
                <input type="text" name="lastName" required minlength="2" />
            </inform-field>
        </label>
        <button type="submit">Submit</button>
    </form>
</inform-el>
```
